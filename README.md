# 🗺️ Spatial Energy Analysis

> Spatial analysis of urban power outage events using Python and geospatial techniques.

---

## 📌 About the Project

This repository contains a spatial data analysis workflow developed to explore urban power outage events using geospatial analysis techniques.

The project combines Python and GIS tools to identify spatial patterns, visualize geographic distributions and apply exploratory spatial statistics.

---

## 🚀 Technologies

- Python
- GeoPandas
- Pandas
- Matplotlib
- GeoDA
- PySAL

---

## 📊 Main Topics

- Spatial Join
- Choropleth Maps
- Moran's I
- LISA Cluster Maps
- Spatial Statistics
